Welcome

Welcome to Apache jUDDI!

jUDDI is an open source implementation of OASIS's Universal Discovery Description and Integration (UDDI). You've reached the splash page for jUDDI web server.

View the jUDDI User Interface - This is a nearly complete UDDIv3 end user web application.

View the jUDDI Administration Interface - This is for administrators to use to reconfigure jUDDI, check the status, and perform administrative actions.

Tomcat Administration Interface - This is for administrators to use to reconfigure Tomcat, check the status, and perform administrative actions.
